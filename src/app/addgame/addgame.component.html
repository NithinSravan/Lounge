Welcome to Lounge Gaming.

Follow these steps to upload your JS game:

  1. Code the game according to your ideas and write a function which is called after the game ends.
  2. This function MUST include parent.postMessage() function to send the score to Lounge servers.
  3. Make sure you don't store your scores or related data in the LocalStorage.
  4. Wrap your game files in a folder with the game's name(NOTE: folder name should be in lowercase without spaces).
  5. Zip your game folder using desired application.
  6. Click "Add Game" to upload your zip file, enter the name of the game(for users to identify) and press "Upload" after ensuring the file you have chosen.
  7. Voila! Your game is now available in Lounge GameHub or everyone to enjoy!
